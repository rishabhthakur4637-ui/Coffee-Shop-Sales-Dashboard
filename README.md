# ☕ Coffee Shop Sales Dashboard

## 📌 Project Overview
This project presents an interactive Coffee Shop Sales Dashboard created using Microsoft Excel. The dashboard helps analyze sales performance, customer footfall, product categories, store locations, and time-based sales trends.

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

---

## 📊 Key KPIs
- Total Sales
- Total Orders
- Total Footfall
- Average Bill per Person
- Average Orders per Person

---

## 📈 Dashboard Features
- Sales Trend Analysis
- Store Location Performance
- Product Category Analysis
- Hourly Sales Analysis
- Weekday vs Weekend Sales
- Interactive Filters using Slicers
